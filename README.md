#  BackHackers AI

## Overview
BackHackers AI is a **fraud detection dashboard** designed to visualize and identify suspicious transactions in real time.  
It provides a clean and interactive interface to monitor transaction data and detect potential fraud patterns.

- **Deployment:** GitHub Pages, Vercel

-  Interactive dashboard UI  
-  Fraud / Safe transaction indicators
